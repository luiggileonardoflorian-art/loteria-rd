# 🎰 LoteríaRD

Portal completo de resultados de loterías dominicanas, construido con Next.js 14.

## Características

- ✅ Resultados verificados en tiempo real
- 🎯 Comprobador de jugadas
- 📊 Estadísticas: números calientes/fríos
- 📅 Histórico de sorteos
- 🕐 Horarios de todas las loterías
- 🎮 Sección Jugar Online (redirige a operadores oficiales)
- 📱 Diseño responsive mobile-first
- ⚡ Optimizado para Core Web Vitals y SEO
- 💰 Espacios AdSense listos

## Stack

- **Framework**: Next.js 14 App Router
- **CSS**: Tailwind + CSS custom (tema oscuro profesional)
- **Fonts**: Syne + DM Sans

## Desarrollo local

\`\`\`bash
npm install
npm run dev
\`\`\`
